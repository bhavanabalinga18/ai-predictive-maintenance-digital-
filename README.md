
# AI Predictive Maintenance Digital Twin

## Overview
This project uses Machine Learning and Digital Twin concepts for predictive maintenance of industrial drilling machines.

## Features
- Failure Prediction
- Tool Wear Monitoring
- Temperature Analysis
- Vibration Monitoring
- RPM Tracking
- Real-Time Dashboard
- Predictive Analytics

## Technologies
- Python
- NumPy
- Pandas
- Scikit-Learn
- Flask
- HTML/CSS/JavaScript

## Dataset
Industrial drilling machine sensor dataset containing:
- Temperature
- Vibration
- RPM
- Tool Wear
- Failure Status
